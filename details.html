SKOOL Project Details

🚧 Project In Progress

1. Idea

SKOOL is a modern learning platform designed for 1BAC, 2BAC, and concours students in Morocco. It solves the problem of unorganized learning by combining video lessons, PDF summaries, QCM quizzes, and progress tracking in one structured system. Instead of using scattered resources like YouTube, PDFs, or WhatsApp groups, students get a clear, guided learning experience. Future updates may include AI tutors, chat systems, certificates, and gamification features. The goal is to make learning faster, more organized, and more effective.

2. What the Platform Includes

✔ Video lessons (5–10 min)
✔ PDF summaries per chapter
✔ QCM quizzes with instant correction
✔ Progress tracking system
✔ Certificates (future)
✔ AI tutor (future)
✔ Chat system (future)
✔ Gamification system (future)

3. Market Reality

YouTube → free but unstructured learning
Udemy / Coursera → not adapted to Moroccan curriculum
WhatsApp / Discord → unorganized student groups

4. Opportunity

There is a strong gap in structured, curriculum-based learning platforms for Moroccan BAC and concours students. This is a big opportunity for SKOOL.

5. Risk

The project is large in scope because it includes:

• Learning platform (LMS)
• Social system
• AI tutor
• Video system
• Exam system
• Gamification engine

This is why it must start simple (MVP first).

6. Strategy

PHASE 1:
✔ Courses + Videos
✔ QCM system
✔ Progress tracking

PHASE 2:
✔ Certificates
✔ Gamification

PHASE 3:
✔ AI tutor
✔ Personalized learning

PHASE 4:
✔ Live classes
✔ Teacher marketplace

7. Conclusion

SKOOL is not perfect yet, but it is a very strong startup idea if you start small, focus on BAC/concours students, and build step by step.
